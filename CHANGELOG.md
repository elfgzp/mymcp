# 更新日志

## [0.1.0] - 2025-11-29

### 新增功能
- ✅ 自定义命令支持（HTTP 和脚本执行）
- ✅ MCP 服务聚合功能
- ✅ 热更新机制（配置变更自动重载）
- ✅ 多种鉴权方式支持
- ✅ Web 管理界面框架
- ✅ uvx 支持（无需安装即可运行）

### 优化
- 🔧 **端口优化**: 默认管理端端口从 8080 改为 18888，避免端口冲突
- 🔧 **端口检查**: 添加端口可用性检查和自动查找备用端口功能
- 🔧 **错误处理**: 改进错误处理和日志记录
- 🔧 **连接管理**: 优化 MCP 客户端连接和断开逻辑

### 修复
- 🐛 修复 MCP 连接初始化问题
- 🐛 修复配置热重载时的资源清理问题

### 技术改进
- 使用 Pydantic 进行配置验证
- 使用 watchdog 进行配置文件监控
- 异步 I/O 支持
- 更好的日志记录
